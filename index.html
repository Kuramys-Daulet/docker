Docker Compose арқылы
CMS орнату

Кез келген кодты бір рет басу арқылы көшіріп алыңыз. Қадамдарды бірізді орындаңыз.

WordPress :8080 Joomla :8082 OpenCart :8080

Терминалда орындаңыз

1
Қалта жасаңыз
mkdir wordpress-docker cd wordpress-docker
2
Файл жасаңыз
notepad docker-compose.yml
Mac / Linux: nano docker-compose.yml
3
Іске қосыңыз
docker compose up -d
4
Браузерде ашыңыз
http://localhost:8080

docker-compose.yml — мазмұнын осы файлға қойыңыз

docker-compose.yml 
version: '3.8'
services:
  wordpress:
    image: wordpress:latest
    ports:
      - "8080:80"
    environment:
      WORDPRESS_DB_HOST: db
      WORDPRESS_DB_NAME: wpdb
      WORDPRESS_DB_USER: wpuser
      WORDPRESS_DB_PASSWORD: pass
    depends_on: [db]
  db:
    image: mysql:latest
    environment:
      MYSQL_DATABASE: wpdb
      MYSQL_USER: wpuser
      MYSQL_PASSWORD: pass
      MYSQL_ROOT_PASSWORD: root_password_here
    volumes:
      - db_data:/var/lib/mysql
volumes:
  db_data: {}

docker-compose.yml — мазмұнын осы файлға қойыңыз

docker-compose.yml 
version: '3.8'
services:
  opencart:
    image: aamservices/opencart:latest
    ports:
      - "8080:8080"
    environment:
      OPENCART_HOST: localhost:8080
      DB_SERVER: mariadb
      DB_DATABASE: opencart
      DB_USERNAME: ocuser
      DB_PASSWORD: ocpass
    depends_on: [mariadb]
  mariadb:
    image: mariadb:10.6
    environment:
      MARIADB_DATABASE: opencart
      MARIADB_USER: ocuser
      MARIADB_PASSWORD: ocpass
      MARIADB_ROOT_PASSWORD: root
    volumes:
      - oc_db:/var/lib/mysql
volumes:
  oc_db: {}

Пайдалы командалар

▶

▶
Логтарды қарау
docker compose logs -f
▶
Іске қосылған контейнерлер
docker ps
■
Деректермен бірге жою
docker compose down -v
